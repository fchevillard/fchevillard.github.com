Step 1 - Chose your drug

Glivec Imitrex Norvasc

Step 2 - Ligand structure

  • Go to the ZINC → Search → Structure
  • Draw the drug into the white panel
  • Save the SMILES code that appears on the right side of the panel "1."
  • Select identity on the menu on the right of the SMILES code → Run Query
  • Among the structures that appears, verify if the first one has the correct ZINC id

    • Glivec Imitrex Norvasc
    ZINC19632618 ZINC00014360 ZINC19796039

    Step 3 - Ligand bioactivity

  • Go to molinspiration → Copy the SMILES code → Calculate Properties
  • Discuss the calculated properties: is the drug Lipinski-like ?

    • n.b. nON property refers to the number of H bond acceptors, while nOHNH refers to the number of H bond donors.

  • Come back to molinspiration → Copy the SMILES code → Predict Bioactivity
  • What is the best bioactivity score ? on which target ?
  • Verify bioactivity score
