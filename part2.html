--- layout: page title: Dataset preparation header: Posts By Tag group: navigation --- In order to find better actives for the target, we will look for similar compounds of the drug in different databases and thereby create a dataset of similar compounds. These similar compounds will be the basis we need for ligand optimization.

Step 4 - ZINC

The ZINC database contains around 20 million compounds. You will first have a look in this database.

  • Go to ZINC → Search → Structure
  • Paste the SMILES code of your drug in the box "1."
  • Set the cut-off of similarity to 90% → Run Query
  • How many compounds do you retrieve ?
  • Do the suggested compounds look similar to the drug ?
  • Modify the cut-off for similarity search, so you retrieve approx. 25 compounds.

    • Step 5 - Ligandinfo

    LigandInfo is a java-based tool that allows you to perform similarity screens in 8 databases and clustering studies.

  • Go to ligandInfo → Run ligand.info (top right corner)
  • File → Open → From Text Box
  • Enter the SMILES code of your drug (query) in the box → Open (if the paste option does not work, you can drag & drop the SMILES code)
  • Search → Search in → Harvard ChemBank
  • Set the number of similar molecules to 20 → Profile Search
  • Can you retrieve your drug in this database ?
  • Do you observe any structure changes ?

    • n.b. The similar compounds will appear in a table. Click on a compound to see its structure. The compounds will be sorted by decreasing similarity, so the compounds in the top in the most similar to the query.

  • Remove some results from the search → Select the compounds you want to remove → Table → Remove Molecules → Yes
  • Get familiar with ligandinfo similarity search by repeating the previous steps using different databases
  • Stop when you have around 20 compounds that you like and do feel comfortable with the similarity search

    • n.b. Feel free to explore the different menus, such as Table or Viewer...
