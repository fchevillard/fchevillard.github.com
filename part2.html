Those similar compounds will be the basis we need for the ligand optimization.

Step 4 - ZINC

The ZINC database exhibits around 20 millions compounds. You will first have a look in this database.

  • Go to the ZINC → Search → Structure
  • Paste the SMILES code of your drug in the box "1."
  • Set up the cut-off of similarity to 90% → Run Query
  • How many compounds do you retrieve ?
  • Does the suggested compounds looks similar to the drugs ?
  • Modify the cut-off for similarity search, so you can work with 25ish compounds.

    • Step 5 - Ligandinfo

    LigandInfo is a java-based tool, that allows you to perform similarity screen in 8 databases and clustering studies.

  • Go to ligandInfo → Run ligand.info (top right corner)
  • File → Open → From Text Box
  • Enter the smile code of your drugs in the box → Open (if the paste option do not work, you have to write the SMILES manually...)
  • Search → Search in → Druglikeness NCI
  • Set the number of similar molecules to 20 → Multiple Search
  • Repeat the previous steps in different databases
