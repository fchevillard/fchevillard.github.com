--- layout: page title: Datasets preparation header: Posts By Tag group: navigation --- In order to find better activs on the target, we will look for similar compounds of the drug within differents databases and therefore create a dataset of similar compounds. Those similar compounds will be the basis we need for the ligand optimization.

ZINC

The ZINC database exhibits around 20 millions compounds. You will first have a look in this database.

Basic search

  • Go to the ZINC → Search → Structure

  • Copy the smile code of your drugs in the box called 1.

  • Set up the cut-off of similarity to 90% → Run Query
