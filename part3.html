Step 7 - Clustering

Clustering allows you to regroup similar molecules into a cluster (group of molecules). It can be then interesting to study the outcomes from clustering to see, as instance, within a dataset if a common chemical structure appears often. You will use ligandinfo for this procedure which is based on complete-linkage algorithm.

  • Retrieve the 20 compounds obtained in Part 5
  • Cluster → Normal → Whole Table
  • Set the similarity between molecules to 0.8 → Normal Cluster

    • n.b. The outcomes will appears on the table. The last column (Cluster ID) gives you the ID of the cluster, so you can see which compounds are regroup in the same cluster. Let's then have a closer look to each cluster.

  • Are the compounds within each cluster similar ?
  • Can you find particular features that distinguish two clusters ?
  • Repeat the previous steps and change the similarity values within the cluster.
  • Comment differences.

    • n.b. At some point, you might find a cluster that you like. You can explore similar compounds of the cluster members by doing another similarity search.

  • Select the compounds of the clusters in the table (they should be highlighted in blue)
  • Search → Search in → Whatever Databases you like
  • Set the number of similar molecules to 20 → Multiple Search
  • Comment on the new compounds
