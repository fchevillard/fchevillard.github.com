Step 7 - Clustering

Clustering allows you to (re)group similar molecules into a cluster (group of molecules). It can then be interesting to study the results of clustering calculations in order to check, for instance, whether a common chemical structure appears frequently within a dataset. You will use ligandinfo for this procedure, which is based on the complete-linkage algorithm.

  • Retrieve the 20 compounds obtained in Part 5
  • Cluster → Normal → Whole Table
  • Set the similarity between molecules to 0.8 → Normal Cluster

    • n.b. The outcome will appears in the table. The last column (Cluster ID) gives you the ID of the cluster, so you can easily see which compounds have been regrouped in the same cluster.

  • Are the compounds within each cluster similar ?
  • Can you find particular features that distinguish two clusters ?
  • Repeat the previous steps and change the similarity values within the cluster.
  • Comment on the differences.

    • n.b. At a certain point, you might find a cluster that you like. You can explore compounds similar to the cluster members by doing another similarity search.

  • Select the compounds of the cluster in the table (they should now be highlighted in blue)
  • Search → Search in → Whichever Database you like
  • Set the number of similar molecules to 20 → Multiple Search
  • Comment on the new compounds

    • Step 8 - ADMET Prediction

    ADMET properties are very important in the drug-design process since they describe the fate of a pharmaceutical compound within an organism. Before submitting your compounds to the pharmaceutical experts, you might want to remove the compounds that are likely a "bad drug" so you don't waste the precious time of your colleagues... To do so, you will use Properties Explorerwhich gives a rapid overview of the chemical properties of a molecule and its druglikeness.

  • Go to Properties Explorer
  • Draw the compounds
  • What is the drug-score ? Druglikeness ?
  • Does the compound exhibit any toxicities ?

    • n.b. Red is a bad sign... while green is a good one !
